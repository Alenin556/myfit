# fit

fit‑приложение для питания и тренировок: профиль, расчёт суточной нормы (ккал и КБЖУ), план питания, дневник питания и базовый план тренировок.

## Ключевой функционал

- **Регистрация и вход**
  - Онбординг: имя, ник (опционально), логин/пароль, год рождения, рост/вес, пол
  - Аватар из галереи
  - Выход из сессии и повторный вход (данные профиля остаются на устройстве)

- **Профиль**
  - Просмотр и редактирование данных пользователя
  - Хранение и обновление текущего веса

- **Цели**
  - Выбор цели: набор, поддержание, снижение
  - Для набора/снижения: постановка цели по весу (шаг изменения и диапазон «от/до»)

- **Расчёт суточной нормы**
  - Экран ввода параметров для расчёта (возраст, рост, вес, пол, активность, цель)
  - Расчёт суточной нормы калорий и КБЖУ и отображение сводки

- **План питания**
  - Генерация рациона на день под целевую калорийность
  - «Обновить план» для получения новой вариации меню
  - Экспорт меню на день в PDF и возможность поделиться/распечатать
  - Отправка текстовой сводки плана во внешние каналы (в т.ч. Telegram)

- **Дневник питания («Подсчёт калорий»)**
  - Добавление продуктов с указанием граммовки
  - Автоматический расчёт ккал и КБЖУ и итоги за день
  - Предупреждение при превышении суточной нормы (с подтверждением добавления)

- **Что поесть**
  - Подбор продуктов и граммовок на один приём пищи на основе плана

- **План тренировок**
  - Выбор типа тренировочной программы и просмотр описания/заметок

- **Настройки**
  - Светлая/тёмная тема (или как в системе)
